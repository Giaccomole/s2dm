Backrest

Description

Describes signals related to the backrest of the seat.

Fields
Field Name Description
isLessLumbarSupportSwitchEngaged - Boolean Is switch for less lumbar support engaged.
isLessSideBolsterSupportSwitchEngaged - Boolean Is switch for less side bolster support engaged.
isLumbarDownSwitchEngaged - Boolean Lumbar down switch engaged.
isLumbarUpSwitchEngaged - Boolean Lumbar up switch engaged.
isMoreLumbarSupportSwitchEngaged - Boolean Is switch for more lumbar support engaged.
isMoreSideBolsterSupportSwitchEngaged - Boolean Is switch for more side bolster support engaged.
isReclineBackwardSwitchEngaged - Boolean Backrest recline backward switch engaged.
isReclineForwardSwitchEngaged - Boolean Backrest recline forward switch engaged.
lumbarHeight - UInt8 Height of lumbar support. Position is relative within available movable range of the lumbar support. 0 = Lowermost position supported.
Arguments
unit - Length_Unit_Enum
lumbarSupport - Float Lumbar support (in/out position). 0 = Innermost position. 100 = Outermost position.
Arguments
unit - Relation_Unit_Enum
recline - Float Backrest recline compared to seat z-axis (seat vertical axis). 0 degrees = Upright/Vertical backrest. Negative degrees for forward recline.
Positive degrees for backward recline.
Arguments
unit - Angle_Unit_Enum
sideBolsterSupport - Float Side bolster support. 0 = Minimum support (widest side bolster setting). 100 = Maximum support.
Arguments
unit - Relation_Unit_Enum

Seat

Description

The model of a vehicle's Seat.

Fields
Field Name Description
heatingCooling - Int8 Heating or Cooling requested for the Item. -100 = Maximum cooling, 0 = Heating/cooling deactivated, 100 = Maximum heating.
Arguments
unit - Relation_Unit_Enum
height - UInt16 Seat position on vehicle z-axis. Position is relative within available movable range of the seating. 0 = Lowermost position supported.
Arguments
unit - Length_Unit_Enum
isBackwardSwitchEngaged - Boolean Seat backward switch engaged.
isBelted - Boolean Is the belt engaged.
isCoolerSwitchEngaged - Boolean Cooler switch for Seat heater.
isDecreaseMassageLevelSwitchEngaged - Boolean Decrease massage level switch engaged.
isDownSwitchEngaged - Boolean Seat down switch engaged.
isForwardSwitchEngaged - Boolean Seat forward switch engaged.
isIncreaseMassageLevelSwitchEngaged - Boolean Increase massage level switch engaged.
isOccupied - Boolean Does the seat have a passenger in it.
isTiltBackwardSwitchEngaged - Boolean Tilt backward switch engaged.
isTiltForwardSwitchEngaged - Boolean Tilt forward switch engaged.
isUpSwitchEngaged - Boolean Seat up switch engaged.
isWarmerSwitchEngaged - Boolean Warmer switch for Seat heater.
massage - UInt8 Seat massage level. 0 = off. 100 = max massage. v5.0 - refactored to Seat.MassageLevel
Arguments
unit - Relation_Unit_Enum
massageLevel - UInt8 Seat massage level. 0 = off. 100 = max massage.
Arguments
unit - Relation_Unit_Enum
position - UInt16 Seat position on vehicle x-axis. Position is relative to the frontmost position supported by the seat. 0 = Frontmost position supported.
Arguments
unit - Length_Unit_Enum
seatBeltHeight - UInt16 Seat belt position on vehicle z-axis. Position is relative within available movable range of the seat belt. 0 = Lowermost position supported.
Arguments
unit - Length_Unit_Enum
tilt - Float Tilting of seat (seating and backrest) relative to vehicle x-axis. 0 = seat bottom is flat, seat bottom and vehicle x-axis are parallel. Positive
degrees = seat tilted backwards, seat x-axis tilted upward, seat z-axis is tilted backward.
Arguments
unit - Angle_Unit_Enum
airbag - Airbag
backrest - Backrest
headrest - Headrest
seating - Seating
instanceTag - InCabinArea2x2
