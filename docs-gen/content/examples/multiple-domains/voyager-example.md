# Interactive GraphQL Schema Visualization

Explore the multiple domains model through an interactive [GraphQL Voyager](https://github.com/graphql-kit/graphql-voyager) visualization. This schema demonstrates how **S2DM** enables cross-domain references while maintaining clear domain boundaries.

{{< callout type="tip" >}}
**Usage:** Click and drag to navigate • Click types for details • Use the sidebar to search and explore documentation • Zoom with mouse wheel for different perspectives.
{{< /callout >}}
