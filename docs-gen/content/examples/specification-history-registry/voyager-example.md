# Interactive GraphQL Schema Visualization

Explore the specification history registry model through an interactive [GraphQL Voyager](https://github.com/graphql-kit/graphql-voyager) visualization. This schema demonstrates how **S2DM** manages specification versions and tracks changes over time.
