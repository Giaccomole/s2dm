# Schema Documentation

Explore the seat domain model GraphQL schema documentation. This model demonstrates hierarchical seat components including backrest, headrest, seating surface properties, and comprehensive seat capabilities with VSS compliance.

{{< callout type="tip" >}}
**Usage:** This section provides comprehensive documentation of the GraphQL schema structure, types, and field definitions for the seat domain model.

---

The seat domain model schema is available for analysis and understanding. The GraphQL schema defines the complete structure of seat-related data types, relationships, and capabilities as defined in the Vehicle Signal Specification (VSS).

For detailed exploration of the schema structure, types, and relationships, refer to the generated GraphQL schema files and documentation in the project repository.
